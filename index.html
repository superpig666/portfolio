Hello, there!

Here is my amazing journey through the UI/UX course.

I am a student at Brown University, and I took a UI/UX class this semester. In the course I learned a lot about UI/UX and made many hands-on designs that I hope will improve people's experience. Here are the details of them.

Personas & Storyboarding

For my first project on Personas and Storyboarding, I chose the Coca-Cola freestyle drink machine in the Andrew Commons cafeteria as the object of observation, observed the user's reaction when interacting with it, and described the corresponding Personas and Storyboard.

Responsive Redesign

My second project was about responsive design. I was browsing the web and noticed that the website of a local dentist in Rhode Island was a bit outdated and not responsive at all, so I used BootStrap to redesign a responsive version of the homepage.

Iterative Redesign

My third project was about iterative design, which was a group assignment. We chose Dumme, a startup about video search, to design the interface on Figma using their ideas, and iterated on the changes after getting feedback.

React Development

My last project was about React development. I designed a bakery website using React and implemented real-time filtering, sorting, and shopping cart functionality through State.
